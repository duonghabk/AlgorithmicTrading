Important Note - Pandas Datareader


In order to continue using pandas datareader to get data from yahoo finance, you need to install older versions of the following libraries.

pandas datareader version <=0.5.0

pandas version <=0.24

spyder version <=3.9

python version <=3.7


Despite installing above libraries you may still get error. Please refer the Q&A section under Pandas Datareader lecture video where troubleshooting steps have been discussed in detail.
